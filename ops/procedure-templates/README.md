# Procedure Templates

These are minimal procedure templates for downstream repos. Customize them with your project details.
They are not this repo's canonical operator procedures.

In this repo:
- canonical operator runbooks live under `workbook/ops/`
- this `ops/procedure-templates/` directory is template-oriented only

Suggested files:
- `deploy-template.md`
- `handoff-template.md`
- `govern-template.md`
- `audit-template.md`

Each concrete runbook should reference:
- `docs/ops-lanes-agent.md`
- `docs/opsec-ops-lanes-signer-map.md`
