# Runbooks (templates)

These are minimal runbook templates for downstream repos. Customize them with your project details.
They are not this repo's canonical operator procedures.

In this repo:
- canonical operator runbooks live under `workbook/ops/`
- this `ops/runbooks/` directory is template-oriented only

Suggested files:
- `deploy.md`
- `handoff.md`
- `govern.md`
- `audit.md`

Each runbook should reference:
- `docs/ops-lanes-agent.md`
- `docs/opsec-ops-lanes-signer-map.md`
