# Artifacts

This directory is not a default public dump zone for live operator outputs.

Keep live artifacts local or in controlled CI storage.
Only commit:
- `README.md`
- `.gitkeep`
- `*.example.*`
- `*.redacted.*`
- deliberately reviewed public-safe fixtures

`artifacts/<network>/current/fe-release/` is also local/generated by default. Treat exported FE handoff bundles the same way unless you deliberately review and publish a public-safe fixture.
